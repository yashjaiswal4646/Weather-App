# üöÄ Core Features

Real-Time Weather Data
Current Temperature: Displayed prominently with Celsius/Fahrenheit options

Weather Conditions: Clear, cloudy, rainy, snowy, stormy with descriptive text

Location Details: City name, country, and local time

Weather Icons: Dynamic SVG icons representing current conditions

Feels Like Temperature: Perceived temperature based on humidity and wind


Detailed Weather Metrics

Humidity Levels: Percentage-based humidity readings

Wind Speed & Direction: Comprehensive wind data with directional indicators

Atmospheric Pressure: Barometric pressure in hPa

Visibility Range: Distance visibility in kilometers

UV Index: Solar radiation intensity levels

Dew Point: Temperature at which condensation occurs

Sunrise & Sunset: Local times for solar events


Extended Forecast

5-Day / 3-Hour Forecast: Detailed future weather predictions

Daily Summaries: High/low temperatures and conditions

Hourly Breakdown: Temperature trends throughout the day

Precipitation Probability: Chance of rain/snow


Search & Location Features

City Search: Global city database with autocomplete suggestions

Geolocation: One-click weather for user's current location

Recent Searches: Quick access to previously viewed locations

Favorite Cities: Save frequently checked locations

Multiple Units: Toggle between metric and imperial systems
